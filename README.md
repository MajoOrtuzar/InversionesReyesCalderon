# Inversiones Reyes Calderón – Sitio Web Corporativo

## Descripción del proyecto

Este proyecto consiste en el desarrollo de un **sitio web corporativo tipo brochure** para *Inversiones Reyes Calderón*.
El objetivo es presentar la empresa, sus servicios y canales de contacto mediante una interfaz clara, moderna y adaptable a distintos dispositivos.

El sitio está diseñado siguiendo principios de **UX/UI y desarrollo frontend**, priorizando una estructura simple, navegación clara y una estética basada en una **paleta de grises** para visualizar la jerarquía de los contenidos antes de aplicar la identidad visual final.

## Objetivo

El objetivo principal del proyecto es crear una **base estructural funcional y responsive** que permita a la empresa:

* Presentar sus servicios de forma clara
* Mostrar contenido corporativo
* Integrar redes sociales
* Facilitar el contacto con clientes potenciales

Además, el proyecto busca implementar buenas prácticas de **maquetación web moderna**, utilizando tecnologías nativas sin depender de frameworks externos.

## Tecnologías utilizadas

El sitio fue desarrollado utilizando tecnologías fundamentales del desarrollo frontend:

* **HTML5** – estructura semántica del sitio
* **CSS3** – estilos y diseño visual
* **CSS Grid y Flexbox** – sistema de layout responsivo
* **Media Queries** – adaptación a distintos tamaños de pantalla

No se utilizan frameworks como Bootstrap para priorizar el aprendizaje y control completo del layout.

## Secciones del sitio

El sitio se estructura en distintas secciones que permiten organizar la información de manera clara:

* **Header**
  Contiene el logo y navegación principal.

* **Hero / Banner principal**
  Sección introductoria con mensaje principal de la empresa.

* **Nosotros**
  Presenta la historia y valores de la empresa.

* **Servicios**
  Muestra los servicios ofrecidos mediante tarjetas informativas.

* **RRSS (Redes sociales)**
  Grid visual inspirado en publicaciones de Instagram.

* **Footer**
  Información de contacto y enlaces relevantes.

## Diseño responsive

El sitio fue diseñado bajo un enfoque **mobile-friendly**, adaptándose a diferentes dispositivos:

* **Desktop** – layout de múltiples columnas
* **Tablet** – reducción de columnas
* **Mobile** – estructura vertical

Esto se logra utilizando **CSS Grid, Flexbox y media queries**.
